# Infrastructure Resource Planning & Portfolio Analytics Dashboard

## Business Problem

Infrastructure organizations must balance project demand against limited engineering and project management capacity.

This dashboard provides executives and portfolio managers with visibility into:

- Resource demand
- Capacity constraints
- Portfolio allocation
- Forecasted shortages
- Project prioritization

## Tools

- Power BI
- DAX
- Power Query
- Excel

## Key Metrics

- Resource Utilization
- Capacity Gap
- Portfolio Demand
- Project Priority Mix
- Budget Allocation

## Dashboard Pages

### Executive Overview
High-level KPIs and portfolio health.

### Resource Capacity Planning
Demand forecasting and staffing analysis.

### Portfolio Analytics
Budget, priorities, project status, and manager workload.
